## Feature Blocks

### 1. Typed Flow Control

Encode gate, rollback, retry, pause, resume, pivot, and refine as explicit workflow state instead of hidden agent behavior.

### 2. Diagnosis-Driven Experiments

Run experiments with structured failure diagnosis, repair actions, and best-result promotion rather than blind retries.

### 3. Grounded Metrics

Bind every reported number to run artifacts and verified metric entries before it reaches a draft.

### 4. Operator Visibility

Inspect live run events, branch comparisons, checkpoints, approvals, and provider health from one console.

### 5. Human Checkpoints

Pause where research judgment matters. Let humans choose direction, approve critical freezes, and inject constraints without breaking the flow.

### 6. Drafts With Evidence Discipline

Generate drafts only from frozen evidence, verified citations, and grounded metrics. Unsupported numbers are blocked before publication.
